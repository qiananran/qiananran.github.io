1、工具的准备

  • Git

    下载并安装Git，在win10系统下的下载和安装方式很简单，基本都是下一步下一步的向下进行。需要注意的是要将该应用安装在除了系统盘的其他盘。避免系统盘太过于满。最后安装成功的样子如下图所示

  • node.js

    直接在官网中下载长期支持的版本，避免其他不必要的麻烦。然而在win10的安装中还是一样的傻瓜式的安装直接一步一步的向下去安装就可以了。最后的安装成功效果为，在命令窗口中输入node –version命令，看是否出现该node的版本。

  • 编辑器（推荐使用vscode）

    和之前的安装下载方式相同相同，建议将vscode的支持放在鼠标右击上，以方便今后使用

3、使用Hexo在本地构建个人博客
  1. 在自己的电脑盘符中随便新建一个空的文件夹。
  2. 使用命令行打开该文件夹，在该命令窗口中使用命令==hexo init==来构建本地最初始的博客。
  3. 经过耐心的等待后，最初始的博客在本地构建好后使用命令==hexo s==启动一下。
  4. 在执行第三步后会出现locahost：4000，直接在浏览器中访问就可以了。
