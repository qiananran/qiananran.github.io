什么是 mycli

一：什么是 mycli

MySQL基于python的一个工具：mycli

mycli是MySQL的 具有自动补全和语法高亮功能的 命令行界面。

安装mycli 前提：python环境的成功安装 （安装过python的就可以直接进行以下操作）

二：使用实例

三：安装Mycli

注意安装Mycli的前提是要安装并配置好了python

使用快捷键win + R 打开运行窗口，输入cmd打开命令提示符

检查自己电脑中的python环境是否是好的：pip list 或 pip --version

如果没有问题的话，就输入命令：pip install mycli,回车进行安装

四：启动

注意要先安装好MySQL数据库，在进行使用

使用快捷键win + R 打开运行窗口，输入cmd打开命令提示符

在命令行提示符中输入：mycli -uroot -p数据库密码
